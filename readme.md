Ver2.2 은 Ver2.1 에서 다음 사항을 개선 
1) 기존에는 Single Awning을 위해 Micro Switch를 한 개만 사용하였으나. Awning이 전방(Front)과 후방(Rear)에 각 1개씩 설치되는 Dual Awning으로 확장하여 Micro Switch를 두 개 사용
2) login 화면의 logo 가운데에 "여기 눌러서 로그인!" 이라는 초록색상의 메세지를 3초 간격으로 잠깐씩 보이게 함.
3) 10분 무활동 시 Manual → AUTO 자동복귀 기능을 임시로 중지 처리(다시 사용하려면 true를 false로 변경)
4) OTA 기능 추가
